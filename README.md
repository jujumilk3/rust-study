# rust-study

## commands
### Base
1. `rustc main.rs` -> main.rs를 컴파일하여 바이너리 생성

### Cargo
1. `cargo --version`
2. `cargo new hello_cargo --bin`
3. `cd hello_cargo`
4. `cargo run` -> 이렇게 하면 바이너리가 생성됨
5. `./target/debug/hello_cargo`
